# Archived Documentation

This directory contains archived and historical files from the Framerr v1.1.6 recovery process.

## Directory Structure

### `/recovered-code/from-memory/`
**Purpose:** Code files and documentation recovered from AI memory during the v1.1.6 source code recovery.

**Contents:**
- 47 recovered JSX, JS, CSS, and markdown files
- Includes recovered widgets, components, settings pages, utilities
- Some files have been migrated to `/src` - these remain as reference

**Status:** Historical reference - do not modify

---

### `/recovery-process/recovery/`
**Purpose:** Documentation of the v1.1.6 recovery process and methodology.

**Contents:**
- Build error tracking and resolution
- File decompression and extraction logs
- Git blob recovery process
- File version analysis and selection
- Recovery strategy documentation

**Status:** Historical record - preserved for future reference

---

### `/pre-recovery-backup/archive-pre-recovery/`
**Purpose:** Backup of files immediately before the recovery migration process began.

**Contents:**
- Archived versions of App.jsx, Sidebar.jsx, and other components
- Subdirectories for common, pages, server, settings, styles, utils, widgets

**Status:** Backup only - not for active use

---

### `/uploaded-from-user/`
**Purpose:** Files uploaded by the user during various sessions.

**Status:** User-provided files - preserved for reference

---

## Important Notes

1. **Do not delete:** All archived files are committed to git for historical reference
2. **Do not modify:** These files are snapshots from specific points in time
3. **Recovery reference:** Useful for understanding what was recovered and how
4. **Code migration:** Check here if looking for original recovered versions of components
