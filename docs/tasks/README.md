# Task Tracking System

This directory contains the task tracking and handoff documentation system.

## Files

- **HANDOFF.md** - Critical context and current state for agent handoffs
- **TASK_CURRENT.md** - Active session work tracking
- **STATUS.md** - Overall project progress dashboard
- **TASK_BACKLOG.md** - Future work queue
- **TASK_COMPLETED.md** - Historical completion log
