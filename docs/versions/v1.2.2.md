# v1.2.2 - 2025-12-19

**Status:** RELEASED

## Summary

Bug fixes, polish, and one new theme. Focused on notification center animations, logging cleanup, and UI refinements.

---

## Changes

### Added
- **Nebula Theme** - Deep space purple, magenta, and cyan accent colors

### Fixed
- **Notification Count Bug** - Read/unread counts now computed directly from notifications array, preventing desync
- **ProxyAuth Log Spam** - Downgraded to DEBUG level, logs only once at startup instead of every request
- **qBittorrent Log Spam** - Removed noisy `server_state` debug logging

### Changed
- **NotificationCenter Polish:**
  - Subtle background tint for unread notifications (`bg-accent/5`)
  - Directional slide animations based on filter navigation
  - Smooth vertical collapse when deleting notifications (`layout="position"`)
  - Animated filter tab indicator (sliding pill)
  - Section headers with counts
- **Sidebar:** "Back to Tabs" button text simplified to "Tabs"
- **Linked Accounts:** Clarified helper text for Overseerr username matching
