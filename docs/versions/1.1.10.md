# v1.1.10

**Released:** 2025-12-13  
**Based on:** v1.1.9

---

## Database Migration System

- Added `server/database/migrator.js` - automatic schema migrations on startup
- Added `server/database/json-utils.js` - utilities for JSON column updates
- Added downgrade detection (refuses to start if DB newer than app)
- Auto-backup before migrations (keeps last 3 backups in `/config/backups/`)

## Internal Improvements

- CHATFLOW v2.0 agent workflow system
  - New `.agent/AGENT.md` master hub
  - Simplified workflows (`/start`, `/end`, `/check`, `/build`)
  - New `docs/reference/` documentation

## Bug Fixes

- SQLite migration stability fixes
  - Database schema initialization on fresh install
  - Custom icon display async/await
  - Permission checks groups format
  - User creation timestamps

## Documentation

- Added `docs/reference/database.md` for database architecture
- Cleanup of legacy documentation (~130 files removed)
